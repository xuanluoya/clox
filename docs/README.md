# clox Documentation

This directory contains documentation for the clox compiler project.

## Documentation Structure

- **Build Documentation**: [build.md](./Build.md) - Build instructions and development workflow
- **API Documentation**: [doxygen/html/index.html](./doxygen/html/index.html) - Auto-generated API documentation using Doxygen
- **TODO List** [TODO.md](./TODO.md) - List of tasks to be completed.

## API Documentation (Doxygen)

The project includes comprehensive API documentation generated by Doxygen:

### Accessing Documentation

Open `docs/doxygen/html/index.html` in your web browser

- **Core**: Bytecode chunks, memory management, values
- **Compiler**: Scanner, parser, and compilation logic
- **VM**: Virtual machine and instruction dispatch
- **Utils**: Debugging, error handling, dynamic arrays

### Generating Documentation

See [build.md](./build.md) for instructions on regenerating the documentation.
